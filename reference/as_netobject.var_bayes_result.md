# Coerce a var_bayes_result to plottable netobjects

Coerce a var_bayes_result to plottable netobjects

## Usage

``` r
# S3 method for class 'var_bayes_result'
as_netobject(x, ...)
```

## Arguments

- x:

  A `var_bayes_result`.

- ...:

  Ignored.

## Value

A `netobject_group` with `temporal` (directed) and `contemporaneous`
(undirected) netobjects.
