# Print a list of per-subject ordinary VARs

Prints a concise summary of subject count, variable count, lag-pair
counts, and temporal edge counts for a
[`build_var_each()`](https://saqr.me/idiographic/reference/build_var_each.md)
result.

## Usage

``` r
# S3 method for class 'var_list'
print(x, ...)
```

## Arguments

- x:

  A `var_list`.

- ...:

  Ignored.

## Value

`x`, invisibly.
